## 1. 프로젝트 개요

### 1.1 목적
입학관리팀이 한글(.hwp/.hwpx)·엑셀(.xlsx) 파일을 일일이 열지 않고도,
사내망 또는 내부 정적 호스팅에서 **2026학년도 수시모집 통계**와
**2025학년도 입시결과 통계**를 빠르게 열람·검색·비교할 수 있는 웹페이지를 구축한다.

### 1.2 핵심 원칙
1. **정적 사이트**: 서버 없이 HTML/CSS/JS만으로 동작 (USB·사내공유폴더·GitHub Pages·Netlify·Vercel 어디든 배포 가능)
2. **데이터-뷰 분리**: 원본(.hwpx/.xlsx)은 그대로 두고, 변환 스크립트로 `data/*.json` 산출
3. **점진적 개발**: 2025·2026 자료가 아직 작업 중 → 더미/샘플 JSON으로 먼저 구조를 만들고, 데이터가 채워지면 자동 반영
4. **모바일 보조 대응**: 내부용이지만 태블릿·노트북 양쪽에서 표가 깨지지 않도록 반응형 처리

### 1.3 산출물 (최종)
- `site/` 정적 웹사이트 (HTML·CSS·JS)
- `data/` 자동 생성된 JSON 데이터셋
- `scripts/` HWPX/XLSX → JSON 변환 파이썬 스크립트
- `README.md` 운영 매뉴얼 (데이터 갱신 방법, 빌드 방법)

## 5. 데이터 변환 파이프라인

### 5.1 HWPX → JSON
- **라이브러리**: `python-hwpx` 또는 직접 ZIP 풀어서 OWPML 파싱 (`<hp:tbl>` 요소)
- **절차**:
  1. `hwpx`는 ZIP → `Contents/section0.xml` 등에서 `<hp:tbl>` 추출
  2. 각 표를 `headers`/`rows` 2차원 배열로 변환
  3. `2026학년도_수시모집_통계표_목록.xlsx` 의 80개 메타와 **순서 매칭** (연번 기준)
  4. 파일명 규칙: `{대분류번호}-{중분류번호}-{연번}_{슬러그}.json`

### 5.2 HWP(.hwp 구버전) → JSON
- **옵션 A (추천)**: `LibreOffice --headless --convert-to hwpx`로 1차 변환 후 5.1 흐름 사용
- **옵션 B**: `pyhwp` (CLI: `hwp5txt`, `hwp5html`) → HTML 추출 → BeautifulSoup으로 표 파싱
- 2025 자료는 작업 완료 후 한 번 변환하면 충분 (자주 안 바뀜)

### 5.3 XLSX → JSON
- **라이브러리**: `openpyxl` (이미 사내 환경에서 사용 중일 가능성 높음)
- 시트별 표 헤더 자동 인식 (다중 헤더 행 → 평탄화)
- `최근 2년 모집인원 및 지원자.xlsx`는 `data/compare/2025_vs_2026.json`으로 별도 산출

### 5.4 빌드 명령
```bash
# 한 번에 전체 변환
python scripts/build_all.py

# 개별 변환
python scripts/hwpx_to_json.py "2026학년도 수시모집 통계작업.hwpx" --year 2026 --out data/2026/
python scripts/xlsx_to_json.py "최근 2년 모집인원 및 지원자.xlsx" --out data/compare/2025_vs_2026.json
python scripts/build_index.py --year 2026
```

### 5.5 갱신 워크플로우 (사용자 수동)
1. 한글에서 `2026학년도 수시모집 통계작업.hwpx` 수정·저장
2. 명령창에서 `python scripts/build_all.py` 실행
3. `site/index.html` 새로고침 → 자동 반영 (캐시 버스팅 위해 `?v=날짜` 권장)

- 좌측 사이드바: 대분류·중분류·소분류 트리 (접고 펴기)
- 상단 검색바: "예: 의예과", "잠재역량", "정원내" 등 통합 검색
- 메인:
  - 선택된 표 제목 + 메타(원본 위치·갱신일)
  - 표 본문 (정렬·필터 가능)
  - 자동 생성 차트 (표 성격에 따라 막대/도넛/히트맵 선택)
  - [CSV 다운로드] [인쇄] 버튼

- 좌·우 2열 레이아웃 (2025 / 2026)

- 상승 5개·하락 5개 자동 강조
- 종합 차트: 모집인원·지원자·경쟁률 비교 막대(2개씩 묶음)

### 6.4 `table.html?id=...` (단일 표 영구링크)
- 즐겨찾기·공유용 단일 표 페이지
- URL 파라미터로 특정 표 직접 접근
